Company

Details about Arcane Labs.

Arcane Labs is an independent software studio based in Bengaluru, India. It serves as the publishing home for applications, libraries, and developer tools built around the principles of privacy-first and local-first software.

The label will be registered as an independent company in India during 2026. Until formal registration is complete, Arcane Labs operates as the working name under which all published work is released.

── FOUNDER ────────────────────────────────────────────────

Arcane Labs is founded by Ajit Kumar Gunturi, an R&D software engineer with over a decade of experience designing scalable, resilient systems on cloud-native platforms — platform and control-plane engineering, distributed systems, API strategy. ISB Emerging Leaders alum.

── VALUES ─────────────────────────────────────────────────

Three rules govern what gets built here:

  • Your data should stay yours. Encryption boundaries, client-side processing, user-owned storage — not a toggle, not a policy.
  • The best features don't need a server. Apps that work on an airplane, cost nothing to run, and don't depend on someone else's uptime.
  • Complexity should live in the architecture — not in the user's face.
