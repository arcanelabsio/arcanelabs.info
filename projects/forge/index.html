forge.

One CLI, four AI coding assistants.

A command-line tool that installs engineering-quality plugins into GitHub Copilot, Claude Code, Codex, and Gemini. A plugin is authored once, then rendered for each assistant by its adapter. The installed files are strictly read-only and call gh directly from the user's terminal — no bundled runtime, no proxy, no cached answers.

Install

npx forge-ai-assist@latest

Published on npm. Requires Node 22 or newer. MIT licensed. Forge asks which assistants you use, installs the plugins, and you're done.
